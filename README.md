# The traveling salesman problem with Simulated Annealing (Euclidean distance)

- HTML Canvas, CSS3, JavaScript, Webpack, yarn

## Usage

### Installation

Install the dependencies

```sh
$ yarn install
```

### Serve
To serve in the browser 

```sh
$ yarn start
```

### Build
Compile and build

```sh
$ yarn build
```
